搭建方法

用github账户新建一个仓库，命名为：username.github.io
默认安装node.js 打开cmd
node -v
npm -v
安装git
任意路径任意名字新建文件夹
进入新建文件夹打开git Bash 命令行窗口
输入命令:
npm install hexo -g
输入命令
hexo init
生成静态页面
hexo generate
本地启动
hexo server

浏览器输入 http://localhost:4000

修改 _config.yml 输入命令
vim _config.yml
去到最后一行修改你的github仓库地址
deploy:

 type: git

 repo: https://github.com/yourname/yourname.github.io.git

 branch: master
执行命令
npm install hexo-deployer-git --save
执行命令
hexo deploy
访问你的github仓库地址
yourname.github.io
